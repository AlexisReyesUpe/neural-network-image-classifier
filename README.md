# 🧠 CNN Image Classifier

Proyecto de red neuronal convolucional (CNN) para clasificación de imágenes desarrollado en Python utilizando Google Colab.

## 🚀 Tecnologías
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## 📌 Descripción
Este proyecto implementa una red neuronal convolucional para clasificar imágenes, entrenando un modelo capaz de identificar diferentes categorías.

## 🧠 Funcionalidades
- Preprocesamiento de imágenes
- Entrenamiento de modelo CNN
- Evaluación de precisión
- Visualización de resultados

## ▶️ Ejecución
1. Abrir el archivo `.ipynb` en Google Colab
2. Ejecutar las celdas paso a paso
